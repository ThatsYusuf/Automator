# Automator

A desktop automation application built with Python that enables users to record and replay keyboard and mouse macros, configure an autoclicker, and save automation profiles.

## Features
1.Macro recording
2.Macro playback
3.Configurable autoclicker
4. Global hotkeys
5.Save/load automation profiles
6.Live telemetry dashboard


## Tech Stack

1. Python
2. CustomTkinter
3. pynput
4. JSON
5. threading
6.  logging

## Installation

pip install -r requirements.txt

python main.py
