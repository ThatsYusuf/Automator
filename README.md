# Automator
A macro that repeatedly either clicks a certain key or set of keys.

Includes:
1. A configurable autoclicker
2. Save Files for having macro recordings
3. Live Dashboard
4. Global Hotkeys

Built with:
1. Python
2. CustomTkinter
3. pynput
4. JSON
5. Threading
